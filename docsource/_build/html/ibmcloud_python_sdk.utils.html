ibmcloud_python_sdk.utils.common.query_wrapper(conn_type, method, path, headers=None, payload=None)¶

Execute HTTP query and return JSON response. :param conn_type: Define which URL should be used for the connection

such as “iaas”, “auth”, “cis”, or “rg” (resource group).

Parameters

  • method – HTTP method that should be used such as GET, POST, PUT, DELETE, etc…

  • path – Path used by within the query

  • headers – Optional. Headers to send with the query is required such authentication token, content type, etc…

  • payload – Optional. JSON payload send during the query.
